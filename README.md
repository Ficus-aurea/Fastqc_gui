#  FastQC Pro (Lite Version)

Графическое приложение для анализа качества данных секвенирования (формат FASTQ).


##  Особенности

*   **Современный интерфейс:** Темная тема (Dark Mode) на базе `ttkbootstrap`.
*   **Анализ файлов:** Поддержка выбора файлов `.fastq` и `.fastq.gz` через системный диалог.
*   **Визуализация данных:**
    * **Сводка:** Общая статистика (количество чтений, длины, GC-состав).
    * **Длины:** Гистограмма распределения длин прочтений.
    * **Качество:** График среднего качества (Phred score) по позициям (с цветовыми зонами: плохое/среднее/хорошее).
    * **Нуклеотиды:** График процентного содержания A, T, G, C по позициям.

##  Технологии

*   **Python**
*   **Tkinter** (базовый GUI)
*   **ttkbootstrap** (современные стили и виждеты)
*   **Matplotlib** (построение графиков)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Ficus-aurea/Fastqc_gui.git
```

```bash
cd Fastqc_gui
```


### 2. Установка зависимостей

```bash
pip install -r requirements1.txt
```

### 3. Запуск приложения

```bash
python fastqc_gui.py
```

## Руководство пользователя

1.  Нажмите кнопку **"Открыт FASTQ файл"** в левом верхнем углу.
2.  Выберите файл формата `.fastq` (или любой другой для тестовой генерации).
3.  Дождитесь окончания анализа (курсор изменится на "часы").
4.  Переключайтесь между вкладками для просмотра результатов:
    *   **Сводка:** Текстовый отчет.
    *   **Длины:** График разброса длин чтений.
    *   **Качество:** Оценка качества секвенирования.
    *   **Нуклеотиды:** Баланс нуклеотидов.
